Welcome to Kanhu!

A Q&A website based on

Express, Mongodb, Bootstrap, jQuery

Features v0.1

  1. Post questions
  2. Answer asker's questions
  3. 'Like' or 'Dislike' answers
  4. Show user's profile

Get started

  1. Download Mongodb and open it by
    2. cd /your/mongodb/path/bin
    ./mongod --dbpath ~/your/mongodb/path/data/db
  2. Run the following commands in your terminal
    3. cd /your/project/path
    npm install
    npm start
  3. Open your browser and input:
    4. localhost:3000

BUG report

Website stays in loading status for a while when 'like' or 'dislike' is pressed too fast, which sometimes causes the server shutting down.
