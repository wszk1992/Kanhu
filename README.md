# Kanhu
A Q&A web project.
##Usage
1. Download [Mongodb](https://www.mongodb.com/) and open it as database in PORT: 27017  
2. Input these commands in your terminal  

`cd /your_project_path`  
`npm install`  
`npm start`  
3. Open your browser and input the following url in address bar  
`localhost:3000`
